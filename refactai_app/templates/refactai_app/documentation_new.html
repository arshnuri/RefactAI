🚀 Quick Start Guide

Get up and running with RefactAI in minutes

📥

1. Clone & Setup

Download RefactAI and set up your environment with automated scripts.

git clone https://github.com/arshnuri/RefactAI.git cd RefactAI python setup.py
🔑

2. Configure API

Set up OpenRouter API for the best refactoring experience.

echo "OPENROUTER_API_KEY=your_key_here" > .env echo "DEFAULT_MODEL=anthropic/claude-3.5-sonnet" >> .env
🎯

3. Launch Interactive CLI

Start the menu-driven interface for easy refactoring workflows.

python cli.py
🚀

4. Refactor & Push

Use the "Refactor and Push to Git" option for seamless code improvement.

🆕 New Feature: Select files from any drive, refactor with AI, and push to Git in one workflow!

🔗 Git Integration

Seamless version control with automated refactoring hooks

1 Git Hook Setup

Enhanced Git hooks with Unicode fixes and better error handling for all platforms.

🆕 Recent Improvements:
  • ✅ Fixed Unicode encoding issues on Windows
  • ✅ Improved error handling and recovery
  • ✅ Lightweight implementation with minimal dependencies
  • ✅ Better AST validation integration

Automated Setup:

# Complete setup with hooks python setup.py # Or traditional setup python setup_local_llm.py

Manual Installation:

# Copy enhanced hook (recommended) cp git-hook-ascii.py .git/hooks/pre-push # Or minimal version cp git-hook-minimal.py .git/hooks/pre-push # Make executable (Unix/macOS) chmod +x .git/hooks/pre-push

2 Git Workflow Integration

Normal Git commands with automatic refactoring before pushes.

# Standard Git workflow git add . git commit -m "Your changes" git push # ✨ Automatic refactoring happens here! # Bypass refactoring if needed git push --no-verify # Test hook without pushing git push --dry-run
🔄 Hook Process: When you push, the hook automatically detects modified code files, refactors them using AI, validates syntax with AST, creates backups, and then pushes the improved code.

🤖 API Configuration

Configure AI models for optimal refactoring performance

1 OpenRouter API (Recommended)

Access to multiple state-of-the-art AI models through a single API.

🎯 Why OpenRouter: Best performance, multiple model options, reliable service, and excellent documentation.
# Get API key from https://openrouter.ai/ echo "OPENROUTER_API_KEY=your_key_here" > .env echo "OPENROUTER_API_URL=https://openrouter.ai/api/v1/chat/completions" >> .env echo "DEFAULT_MODEL=anthropic/claude-3.5-sonnet" >> .env echo "PREFER_LOCAL_LLM=false" >> .env # Test connection python test_openrouter_api.py

🔧 Troubleshooting & Fixes

Automated solutions for common issues

🚨 Automated Fix Scripts Available: RefactAI includes self-healing capabilities with automated fix scripts for common issues!

🐛 CLI Validation Errors

Issue: 'ASTValidator' object has no attribute 'validate_syntax'

python fix_ast_validator.py

🔗 Git Hook Issues

Unicode Encoding Errors (Windows):

cp git-hook-ascii.py .git/hooks/pre-push

Permission Issues (Unix/macOS):

chmod +x .git/hooks/pre-push

Bypass Hook Temporarily:

git push --no-verify

  • ✅ Better error handling and recovery
  • ✅ Minimal dependencies
